## Description

```
Practicing with nest js and mysql.
Functionality:
 - login/register with help of jwt token
 - CRUD for users
 - CRUD for posts
 - Roles for users
 - Depends on role user has certain functionality
 - Files functionality (upload and serve static files)
```

## Technologies

```
 - nest js
 - mysql
 - sequelize
 - jwt
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
